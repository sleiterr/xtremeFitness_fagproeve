## Vigtige ændringer

**Ændring af telefonfelt i beskedmodellen**

I filen `lib/db/models/message.model.mjs` blev feltet `phone` ændret fra `Number` til `String`:

Før:

```javascript
phone: { type: Number },
```

Efter:

```javascript
// Changed from Number to String to support phone numbers with +, spaces, and different formats from frontend forms
phone: { type: String },
```

Denne ændring blev lavet, fordi kontaktformularen (`ContactForm.jsx`) ikke kunne sende telefonnummeret til API'et, hvis det indeholdt tegn som +, mellemrum eller andre formater. Nu understøttes alle gængse telefonnumre fra frontend.

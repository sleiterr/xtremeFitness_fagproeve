## Projektkommandoer

```bash
npm run dev      # lokal udvikling
npm run build    # produktionsbuild
npm run preview  # lokal visning af build
npm run lint     # eslint check
```

## Sådan tester du funktionalitet

1. Tjek kontaktformularen (src/components/Contact/ContactForm.jsx).
2. Prøv CRUD for blogs i adminpanelet (src/pages/Backoffice/).
3. Tjek visning af træningsøvelser, services, anmeldelser.

## Kvalitetstjek

- Kør `npm run lint` før aflevering.
- Test responsivt på mobil og desktop.
- Tjek API-integration i src/utils/api.js.

## Publicering af dokumentation

1. Opdater indhold i `docs/`.
2. Tjek via https://docs.page/preview.
3. Push ændringer til repoet.

## Udvidelse af funktionalitet

- Tilføj nye komponenter i src/components/
- Udvid API-endpoints i backend
- Tilpas adminpanelet for flere datatyper

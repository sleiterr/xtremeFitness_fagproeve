## Formål

Projektet er en webplatform for en fitnessklub, der viser hvordan man bygger en moderne SPA med React, Tailwind og API backend. Fokus er på datahåndtering, brugeroplevelse og adminfunktioner.

## Hovedfunktioner

- Forside med klubinfo, øvelser, services, anmeldelser
- Kontaktformular med validering
- Adminpanel med CRUD for blogs
- Statistik og testimonials
- Responsivt layout

## Brugerflow

1. Bruger lander på forsiden (Home)
2. Bruger ser klubinfo, øvelser, services
3. Bruger kan sende besked via kontaktformular
4. Admin logger ind og styrer blogs i Backoffice

## Dataflow

- API-kald via src/utils/api.js
- Backend håndterer data og validering
- Frontend viser data i komponenter

## Struktur

```
xtreme_fitness_materials/
  src/
    components/
    pages/
    utils/
    assets/
  public/
  docs/
```

## Mapper og ansvar

- `src/components/` — UI-komponenter: About, Contact, Exercises, Services, Testimonials osv.
- `src/pages/` — Sider til routing: Home, Backoffice (admin-dashboard).
- `src/utils/` — API-kald, datahelpers.
- `src/assets/` — Ikoner, logo, billeder.
- `public/` — Statisk indhold.
- `docs/` — Dokumentation til docs.page.

## Vigtige filer

- `src/App.jsx` — hovedlayout og routing.
- `src/utils/api.js` — API-funktioner.
- `src/components/Contact/ContactForm.jsx` — kontaktformular.
- `src/pages/Backoffice/Backoffice.jsx` — adminpanel.
- `lib/db/models/message.model.mjs` — beskedmodel (backend).
